StarShort API
Access StarShort short dramas through a unified API: search titles, browse trending and popular feeds, retrieve episode lists with cover art, and resolve HLS streaming URLs via DramaBos CDN.

Authentication required
Every request must include your API key as a Bearer token in theAuthorization header. Requests that are missing a token, or use an invalid, inactive, or expired one, are rejected with401 Unauthorized. Get your key from the dashboard.

curl -X GET "https://prod-api.dramabos.live/starshort/api/v1/search?q=love" \
  -A "Mozilla/5.0" \
  -H "Authorization: Bearer YOUR_TOKEN"
